# HeadyBrowser Quickstart - Web Extension

## Installation
```powershell
cd heady-browser
yarn install
yarn build:chrome  # or build:firefox
```

## Loading Extension
1. Chrome: `chrome://extensions` → Enable developer mode → Load unpacked
2. Select `dist/chrome` directory

## Features
- Right-click context menu actions
- Page analysis with `Alt+H` shortcut
- Real-time collaboration
