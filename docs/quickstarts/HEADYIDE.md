# HeadyIDE Quickstart - Desktop AI Assistant

## Installation
```powershell
cd heady-ide
yarn install
yarn build
```

## Configuration
1. Create `.env` file:
```ini
HEADY_API_KEY=your_api_key
BACKEND_URL=https://api.headyio.com
```

## Usage
```powershell
yarn start
```
- Press `Ctrl+Shift+H` to open command palette
- Use `/buddy` to chat with HeadyAI
- Use `/deploy` for instant deployments
