# Manual Installation

## Desktop Installation
1. Navigate to `headybrowser-desktop\dist`
2. Copy all files to:
   - Admin install: `C:\Program Files\HeadyBrowser`
   - User install: `$env:LOCALAPPDATA\HeadyBrowser`
3. Create shortcut manually if needed

## Mobile Installation
1. Transfer `headybrowser-mobile\android\app\build\outputs\apk\release\app-release.apk` to device
2. Open file manager on device and tap APK
3. Allow "Install Unknown Sources" if prompted
