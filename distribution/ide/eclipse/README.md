# Heady Dev Companion for Eclipse

Java OSGi plugin for Eclipse 2024+.

## Status: Planned

## Planned Features
- Content assist via `ICompletionProposalComputer`
- View/editor for Heady chat panel
- Context menu: Explain, Refactor, Tests, Docs, Fix
- Agent mode for multi-file changes
- Integration with Eclipse Marketplace

## Tech Stack
- Java 17+
- Eclipse Plugin Development Environment (PDE)
- OSGi bundles
- SWT/JFace for UI
- HTTP client to heady-api
