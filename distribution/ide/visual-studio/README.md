# Heady Dev Companion for Visual Studio

VSIX extension for Visual Studio 2022+.

## Status: Planned

## Planned Features
- Inline completions via `IAsyncCompletionSource`
- Tool window with chat panel (WPF/XAML)
- Context menu: Explain, Refactor, Tests, Docs, Fix
- Agent mode for multi-step tasks
- Voice input integration

## Tech Stack
- C# / .NET
- Visual Studio SDK (VSIX)
- WPF for UI panels
- HTTP/WebSocket client to heady-api
