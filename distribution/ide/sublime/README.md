# Heady Dev Companion for Sublime Text

Python plugin for Sublime Text 4+.

## Status: Planned

## Planned Features
- Inline completions via `EventListener.on_query_completions`
- Command palette commands (Explain, Refactor, Tests, Docs, Fix)
- Side panel chat via Phantom/HTML sheets
- Agent mode
- Voice input

## Tech Stack
- Python 3.8+ (Sublime Text embedded Python)
- HTTP client to heady-api
- HTML/CSS for panel rendering
