# Heady + Make (Integromat) Integration

Connect Heady to 1000+ apps via Make scenarios.

## Modules

| Module | Type | Description |
|--------|------|-------------|
| Chat with Heady | Action | Send a message, get AI response |
| Run Heady Agent | Action | Trigger an agent with a task |
| Query Knowledge Base | Action | RAG query against your docs |
| Call MCP Tool | Action | Execute any MCP tool |
| Watch for Events | Trigger | Listen for Heady events via webhook |

## Setup

1. In Make, add "Heady AI" from the app catalog
2. Create a new connection with your API endpoint and key
3. Build scenarios using Heady modules

## Example Scenarios

- **Google Sheets -> Heady -> Email**: Analyze spreadsheet data and email insights
- **Typeform -> Heady Agent -> CRM**: Process form submissions with AI and update CRM
- **Scheduled -> Heady RAG -> Google Docs**: Weekly knowledge report
